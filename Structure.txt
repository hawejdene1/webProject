

LES DOSSIERS CONCERNANTS LE BACKEND
/src : contient les fichiers/dossiers concernant les traitement (acces bases de données, calculs etc..)
/includes/api : contient les fichiers d'interfacage qui offre les services backend au frontend

LES DOSSIERS CONCERNANTS LE FRONTEND
/includes/controller : Tous les controllers du projet
/includes/interface  : Tous les fichiers dont on a besoin pour l'interface (FRONTEND)
/res : contient les fichiers CSS et JS pour l'interface
